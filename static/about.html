About Nextern

An explainable internship recommender: skill-ontology matching, skill-gap coaching, free-text job-description scoring, and an optional Claude-powered resume copilot.

Model card

ApproachContent-based recommendation via a curated skill ontology — alias resolution (JS → JavaScript), skill families for partial credit (React ↔ Vue), coverage scoring per company, plus a small CGPA-proximity nudge (15%).
Training data51 synthetic student–company placements across 11 companies. Small and synthetic by design — this is a portfolio demonstration, not a production system.
EvaluationLeave-one-out: each student is removed from the company requirement profiles before being scored. ~82% top-1, ~100% top-3. Treat as a sanity signal, not a benchmark.
ExplainabilityEvery recommendation returns matched / related / gap skill buckets — nothing is a black box.
AI copilotOptional. Uses the Anthropic API (Claude) to extract a profile from resume text and write a coaching narrative. Only active if the server is configured with an API key; resume text is sent to Anthropic's API when used.
Known limitationsTiny synthetic dataset; ontology covers common tech skills only; CGPA signal is heuristic; recommendations should never be the sole basis for career decisions.

API reference

All endpoints return JSON with an ok flag. Prediction endpoints are rate-limited per IP.

EndpointMethodBodyReturns
/api/predictPOST{"Technical skills", "CGPA"}Top-3 companies with confidence, matched / related / gap skills
/api/match_jdPOST{"Technical skills", "jd_text"}Coverage % against the pasted job description, with the skill breakdown
/api/copilotPOST{"text"}Extracted profile, ranked matches, and an AI coaching narrative (503 if not configured)
/api/insightsGET—Dataset stats, per-company profiles, evaluation metrics
/api/configGET—Feature flags (e.g. whether the copilot is enabled)
/healthGET—Liveness check
Example
curl -X POST https://getnextern.onrender.com/api/predict \
  -H "Content-Type: application/json" \
  -d '{"Technical skills": "python, machine learning", "CGPA": 8.5}'
